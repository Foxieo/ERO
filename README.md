ENGLISH
Elden Ring Optimized (ERO) is a fix for Elden Ring that adjusts the game's process settings to improve performance and eliminate screen tearing. The script automatically detects your processor configuration and applies optimal settings. FPS becomes more stable, and stuttering is fixed. The fix works with any version of the game.
For non-Steam versions:
Place the file in the game's root folder and launch the game only via ERO.bat. If you are using Seamless Co-op, the fix will automatically launch the game through Seamless Co-op.
For Steam:
Place the file in the game's root folder, then in the launch options, add: 
"path to ERO.bat" %command%

Example: 
"D:\Games\ELDEN RING\Game\ERO.bat" %command%

Launching Seamless Co-op:
If you are using Seamless Co-op, the fix will automatically launch the game through Seamless Co-op.
