# ERO
ELDEN RING Optimized

ENGLISH

Elden Ring Optimized (ERO) is a fix for Elden Ring that adjusts game parameters to improve performance. The script automatically detects your processor configuration and applies optimal settings. FPS becomes more stable and stuttering is eliminated. The fix works on any version of the game.

Installation on a No Steam version:
Drop the file into the game's root folder and launch the game only via ERO.bat. If you're playing via Seamless Co-op, the fix will automatically launch the game via Seamless Co-op.

Installation on Steam:
Drop the file into the game's root folder and then enter "path to ERO.bat" %command% in the launch parameters.
Example: "D:\Games\ELDEN RING\Game\ERO.bat" %command%

Launching Seamless Co-op:
If you're playing via Seamless Co-op, the fix will automatically launch the game via Seamless Co-op.

SPANISH

Elden Ring Optimized (ERO) - es una modificación para Elden Ring que ajusta los parámetros del juego para mejorar el rendimiento. El script detecta automáticamente la configuración de tu procesador y aplica los ajustes óptimos. Los FPS se vuelven más estables y se eliminan los tirones. La modificación funciona en cualquier versión del juego.

Instalación en una versión no Steam:
Copia el archivo a la carpeta raíz del juego e inicia el juego solo con ERO.bat. Si juegas en modo cooperativo sin interrupciones, la modificación lo iniciará automáticamente.

Instalación en Steam:
Copia el archivo a la carpeta raíz del juego y luego introduce "ruta a ERO.bat" %command% en los parámetros de inicio.
Ejemplo: "D:\Games\ELDEN RING\Game\ERO.bat" %command%

Seamless Co-op:
Si juegas en Seamless Co-op sin interrupciones, la modificación lo iniciará automáticamente.

JAPANSE

Elden Ring Optimized (ERO) - は、Elden Ring のゲームパラメータを調整してパフォーマンスを向上させる修正プログラムです。スクリプトはプロセッサ構成を自動検出し、最適な設定を適用します。FPS が安定し、カクツキが解消されます。この修正プログラムは、ゲームのどのバージョンでも動作します。

Steam 非対応版へのインストール：
ファイルをゲームのルートフォルダにドロップし、ERO.bat のみを使用してゲームを起動してください。シームレス協力モードでプレイしている場合は、この修正プログラムにより自動的にシームレス協力モードでゲームが起動します。

Steam へのインストール：
ファイルをゲームのルートフォルダにドロップし、起動パラメータに "ERO.bat へのパス" %command% と入力してください。
例："D:\Games\ELDEN RING\Game\ERO.bat" %command%

Seamless Co-op：
シームレス協力プレイでプレイしている場合は、この修正プログラムにより自動的に Seamless Co-op を介してゲームが起動します。
